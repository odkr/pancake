Pancake

Toolkit for writing Lua filters for Pandoc.

Pancake aides with

  • maintaining compatibility accross different versions of Pandoc
  • working with complex data structures
  • error handling
  • string interpolation
  • object-oriented programming
  • file I/O and filesystem interaction
  • metadata parsing

See its documentation for details.

It is lightweight, well-documetend, and has been tested with Pandoc v2.9–v2.18. However, it aims to work in any version ≥ 2.0.4.
